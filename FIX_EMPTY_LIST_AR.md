# 🔍 حل مشكلة "عدم ظهور الموظفين في السجل"

## 📌 المشكلة

جدول الموظفين فارغ ولا يعرض أي بيانات في صفحة HRMS.

## 🔧 الحل - 3 طرق

### ✅ الطريقة الأولى: التحقق من البيانات الموجودة

1. **افتح الملف الجديد:** `check_data.html`
   - اضغط على الزر الأزرق "✓ فحص البيانات"
   - سيخبرك إذا كانت البيانات موجودة أم لا

2. **إذا كانت البيانات موجودة:**
   - انسخ البيانات من الصفحة
   - قم بعمل backup (احفظ نسخة)
   - ارجع إلى صفحة hrms.html

3. **إذا كانت البيانات فارغة:**
   - اضغط الزر الأخضر "📥 تحميل بيانات تجريبية"
   - سيتم إضافة 4 موظفين للاختبار
   - ارجع إلى صفحة hrms.html

---

### ✅ الطريقة الثانية: إضافة موظف جديد يدويًا

1. **افتح صفحة hrms.html**
2. **اذهب إلى تبويب "تسجيل موظف جديد"**
3. **املأ البيانات الأساسية:**
   - ✏️ اسم الموظف (إجباري)
   - ✏️ الرقم الوظيفي (سيتم ملؤه تلقائياً: 1001، 1002، إلخ)
   - ✏️ الجنسية
   - ✏️ الجنس
   - ✏️ الهاتف
   - ✏️ البريد الإلكتروني
   - ✏️ تاريخ الالتحاق

4. **اضغط "💾 حفظ البيانات"**
5. **اذهب إلى تبويب "السجل"**
6. **يجب أن ترى الموظف الجديد هناك**

---

### ✅ الطريقة الثالثة: استعادة نسخة احتياطية

إذا كانت لديك نسخة احتياطية من البيانات:

1. **افتح وحدة تحكم المتصفح (F12)**
2. **اذهب إلى تبويب Console**
3. **اكتب:**

```javascript
// إذا كان لديك ملف JSON محفوظ
const data = [
  {
    emp_number: "1001",
    emp_name: "محمد أحمد",
    emp_national: "سعودي",
    // ... باقي البيانات
  },
];

localStorage.setItem("saden_employees", JSON.stringify(data));
location.reload();
```

---

## 📊 مكان تخزين البيانات

البيانات تُخزن في:

- **localStorage** (التخزين المحلي للمتصفح)
- **المفتاح:** `saden_employees`

للتحقق من البيانات في Console:

```javascript
console.log(JSON.parse(localStorage.getItem("saden_employees")));
```

---

## 🔄 سير العمل الصحيح

```
1. تسجيل موظف جديد
        ↓
2. اضغط حفظ البيانات
        ↓
3. البيانات تُحفظ في localStorage
        ↓
4. الصفحة تنعكش (reload)
        ↓
5. البيانات تُحمل من localStorage
        ↓
6. الموظف يظهر في السجل
```

---

## ⚠️ علامات تحذيرية

- ❌ **البيانات فارغة تماماً** → استخدم بيانات تجريبية
- ❌ **الموظفون يختفون عند التحديث** → قد تكون هناك مشكلة في حفظ البيانات
- ❌ **خطأ في وحدة التحكم** → تحقق من وحدة التحكم (F12 → Console)

---

## 🛠️ خطوات تصحيح سريعة

### إذا لم تظهر البيانات:

1. **فتح المتصفح - F12**
2. **اكتب في Console:**

```javascript
// مسح قد يحل المشكلة
localStorage.clear();
location.reload();
```

3. **ثم أضف بيانات تجريبية:**

```javascript
const sample = [
  {
    emp_number: "1001",
    emp_name: "أحمد محمد",
    emp_national: "سعودي",
    gender: "ذكر",
    phone_number: "0501234567",
    email: "ahmed@saden.com",
    company_job: "مدير",
    dept: "الإدارة العامة",
    join_date: "2020-01-15",
    base_salary: "5000",
    vacation_balance: "18",
  },
];
localStorage.setItem("saden_employees", JSON.stringify(sample));
location.reload();
```
