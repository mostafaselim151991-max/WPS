# نظام سادن الذكي 2026 - دليل الاستخدام

## 🔍 تشخيص مشاكل الداش بورد

### المشاكل الشائعة وحلولها:

#### 1. **القيم تظهر صفر أو غير منطقية**

**السبب:** البيانات في localStorage فارغة أو غير موجودة
**الحل:** استخدم صفحة إضافة البيانات التجريبية

#### 2. **عدد الموظفين غير صحيح**

**السبب:** البيانات من جداول مختلفة غير متزامنة
**الحل:** تأكد من إضافة البيانات في جميع الجداول المطلوبة

#### 3. **الإيرادات تظهر صفر**

**السبب:** لم يتم إدخال قيم المشاريع أو الإنتاج
**الحل:** أضف مشاريع مع قيم مالية

## 🚀 خطوات التشغيل الصحيح:

### الخطوة 1: إضافة البيانات التجريبية

1. افتح `http://localhost:8000/test_data.html`
2. اضغط على "إضافة جميع البيانات التجريبية"
3. انتظر حتى تظهر رسالة "تم إضافة جميع البيانات"

### الخطوة 2: فحص البيانات

1. افتح `http://localhost:8000/debug_storage.html`
2. تأكد من وجود البيانات في جميع الجداول

### الخطوة 3: تشغيل الداش بورد

1. افتح `http://localhost:8000/dash.html`
2. يجب أن تظهر القيم الصحيحة الآن

## 📊 مصادر البيانات

### الإحصائيات الأساسية:

- **القوى العاملة**: `saden_employees` + `saden_marketing_staff` + `saden_final_system`
- **المشاريع النشطة**: `saden_db_v4` (حيث status = 'نشط')
- **كفاءة الأداء**: متوسط كفاءة التسويق والصيانة
- **الإيرادات**: مجموع قيم المشاريع + الإنتاج + الكفالات

### الجداول المطلوبة:

- `saden_employees`: بيانات الموظفين
- `saden_marketing_staff`: فريق التسويق
- `saden_fleet_db`: الأسطول
- `saden_db_v4`: المشاريع (CRM)
- `saden_kofalat_db`: الكفالات
- `saden_scrap_v4`: الخردة

## 🔧 استكشاف الأخطاء

### إذا ظلت المشاكل موجودة:

1. افتح Developer Tools (F12)
2. اذهب إلى تبويب Console
3. ابحث عن رسائل الخطأ
4. تأكد من أن localStorage يحتوي على البيانات الصحيحة

### مسح البيانات وإعادة البدء:

```javascript
// في Console:
localStorage.clear();
location.reload();
```

## 📞 الدعم الفني

إذا واجهت مشاكل، تأكد من:

- تشغيل خادم محلي: `python -m http.server 8000`
- فتح الصفحات من نفس المنفذ
- وجود اتصال بالإنترنت لتحميل Chart.js
